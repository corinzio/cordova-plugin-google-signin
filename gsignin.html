(static) config(successCallback, errorCallback, conf)

The config function creates the api client and try to start the connection with google servers. In the conf parameter it is possible to configure wich informations will be sent back upon succesful login.

Parameters:
Name Type Description
successCallback gsignin.connectSuccess

configuration of google api client ended succesfully and the api is now connected.
errorCallback gsignin.cbError

configuration ended with error.
conf gsignin.ConnectionConfiguration

Configuration Object for the connection see gsignin.ConnectionConfiguration

  • 405 Logout Failure
